# CodSoft Java Programming Internship Tasks

This repository contains Java programs completed for the CodSoft Java Programming Internship.

## Tasks Completed

### Task 1: Number Game
- **Description**: A number guessing game where user guesses a random number between 1-100
- **Concepts Used**: Loops, Random class, Scanner, Conditionals
- **File**: `NumberGame.java`

### Task 2: Student Grade Calculator  
- **Description**: Calculates total marks, average percentage, and assigns grade
- **Concepts Used**: Arrays, Methods, Operators
- **File**: `StudentGradeCalculator.java`

### Task 3: ATM Interface
- **Description**: Simple ATM simulation with withdraw, deposit, check balance
- **Concepts Used**: OOP, Classes, Switch case
- **File**: `ATMInterface.java`

## How to Run
1. Clone this repo: `git clone https://github.com/gayatripkkd/Codsoft-Java-tasks.git`
2. Compile: `javac Task1_NumberGame/NumberGame.java`
3. Run: `java Task1_NumberGame.NumberGame`
